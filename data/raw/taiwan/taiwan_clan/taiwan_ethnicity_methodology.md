# 全台灣 Persona 族群欄位建立方法報告

**版本**：v1.0
**建立日期**：2026-06-18
**負責人**：Project Mirror 資料組
**適用範圍**：全台 22 縣市 persona 資料集（`族群` 欄位）
**用途**：作為「宗親地方組織連結強度」等衍生欄位的族群基礎層；台北版方法論的全台擴充前置作業

---

## 1. 報告目的

台北版方法論（`clan_column_methodology.md`）的第一層「族群基礎分」直接讀取既有的 `族群` 欄位。但要擴成全台版，必須先解決一個前置問題：**每個 persona 的族群該怎麼指派，才能讓各縣市的族群結構符合真實人口分布？**

本報告定義一套可重現的指派方法，把四大族群（閩南／客家／外省／原住民）依縣市實際比例分配給 persona，並交付參數表與 Python 實作。

---

## 2. 核心問題：四大族群的定義不對稱

台灣四大族群中，**只有原住民有客觀法定登記**，其餘三族皆為主觀認同的模糊概念，且各有不同的統計基礎：

| 族群 | 認定基礎 | 是否有官方縣市統計 | 主要資料來源 |
|------|---------|------------------|------------|
| 原住民 | 法定（戶籍登記，《原住民身分法》） | ✅ 精確至縣市／鄉鎮 | 內政部戶政司 |
| 客家 | 主觀認同 + 血緣／淵源（《客家基本法》） | ⚠️ 有調查推估縣市比率 | 客委會抽樣調查 |
| 閩南（福老） | 主觀認同 | ❌ 無 | 學術調查推估 |
| 外省（大陸各省市） | 主觀認同（祖籍／遷台時間） | ❌ 無 | 學術調查推估 |

### 2.1 必須處理的陷阱：客家的兩套數字

客家人口有兩個差距極大的官方數字，**絕對不能混用**：

- **《客基法》寬定義 = 19.8%（466.9 萬人）**：只要具客家血緣或淵源、且自我認同為客家人即計入。此定義允許「複數認同」（一個人可同時認同自己是客家人又是福老人），因此會與閩南重疊。
- **主觀單一認同 ≈ 10.9%**：受訪者在四大族群中只選一個時的結果（見 §4.1）。

Persona 的 `族群` 欄位是**互斥單一值**，因此基準必須採「單一認同」邏輯。若直接套用客委會的 19.8% 縣市比率，客家會被嚴重高估、並與閩南重複計算。本報告的解法見 §5 Step 2（縮放校準）。

## 3. 三組關鍵基準數據（皆取自所選資料夾內檔案）

### 3.1 整體族群比例（主觀單一認同基準）

來源：徐富珍、陳信木，〈臺灣當前族群認同狀況比較分析〉（引內政部國民生活狀況調查），受訪者 4,062 人。

| 族群 | 人數 | 比率 |
|------|------|------|
| 臺灣閩南人 | 3,125 | 76.9% |
| 臺灣客家人 | 441 | 10.9% |
| 大陸各省市人（外省） | 405 | 10.0% |
| 臺灣原住民 | 57 | 1.4% |
| 不知道／很難說 | 34 | 0.8% |
| **合計** | **4,062** | **100.0%** |

> 此為**全國基準**，作為各縣市殘差分配的錨點。實務上原住民改用戶政司精確值（1.4%→2.7%，調查低估）。

### 3.2 原住民縣市分布（法定，精確）

來源：內政部統計處《115 年第 8 週內政統計通報——原住民概況》（114 年底）。

- 全國原住民 **62.9 萬人，占總人口 2.7%**；平地原住民 46.7%、山地原住民 53.3%。
- 縣市**占當地人口比率**前三：臺東縣 38.1%、花蓮縣 30.3%、屏東縣 8.1%。
- 原住民**人口數**前三：花蓮 9.5 萬（占全國原民 15.1%）、桃園 9.0 萬（14.3%）、臺東 7.9 萬（12.6%）。
- 都會縣市占比低：新北 1.6%、臺北 0.8%；連江、澎湖、金門、嘉義市、雲林、新竹市均不及 1%。
- 年齡結構年輕：中位數 34.9 歲（全體國民 45.7 歲）→ 影響台北版第二層年齡加權的族群交互。

> 完整 22 縣市的「占縣市人口比率」直接由戶政司表 2 程式讀入，不在此手抄（見 §6 註）。

## 4. 建構方法（縣市層級，四步驟）

對每一個縣市 c，依序計算四族群比例，總和恆為 1：

### Step 1 — 原住民（法定，最優先）

```
p_原住民(c) = 戶政司該縣市原住民占當地人口比率   # 精確值，不調整
```

### Step 2 — 客家（縮放校準，去除複數認同膨脹）

把客委會寬定義比率乘上一個全國校準係數 `k`，使全國加權後的客家比例回到單一認同水準：

```
k = 目標全國客家單一認同比率 / 19.8%
p_客家_raw(c) = 客委會比率(c) × k
```

- 建議 `k` 取 **0.55**（對應單一認同 ≈ 10.9%）。若團隊偏好保留較高客家能見度，可採折衷目標 14%（`k ≈ 0.71`）——此為可調參數，須在資料集說明中載明採用值。
- 縮放後仍保留「新竹縣 > 苗栗 > 桃園 > 花蓮…」的真實地理梯度，只是整體水位下修。

### Step 3 — 外省（推估，無官方縣市資料）

外省無縣市級官方統計，以全國 10% 為基準，依眷村密度／都會程度做相對調整：

```
p_外省(c) = 10% × 眷村都會係數(c)
```

| 縣市類型 | 代表縣市 | 係數 | 推估外省比率 |
|---------|---------|------|------------|
| 眷村／都會高密度 | 臺北、新竹市、桃園、高雄、臺中、屏東、基隆 | 1.3–1.6 | 13–16% |
| 一般都會 | 新北、臺南、嘉義市、宜蘭、花蓮、臺東 | 0.8–1.0 | 8–10% |
| 農業／離島低密度 | 雲林、嘉義縣、彰化、南投、苗栗、澎湖、金門、連江 | 0.3–0.6 | 3–6% |

> 此層不確定性最高，數值為推估。後續可用國防部老舊眷村改建清冊、各縣市眷村列表精修（見 §8）。

### Step 4 — 閩南（殘差）

```
p_閩南(c) = 1 − p_原住民(c) − p_客家_raw(c) − p_外省(c)
```

閩南為最大宗、定義最模糊，故設為殘差以保證四族總和為 1，並避免三重推估誤差累積。若殘差 < 0（極少數客家高比例縣如新竹縣），則按比例壓縮客家與外省，閩南設下限 0。

## 6. Python 實作

```python
import numpy as np
import pandas as pd

# === 基準參數 ===
HAKKA_RAW = {   # 客委會 110 年寬定義比率
    '新竹縣':67.8,'苗栗縣':62.5,'桃園市':39.9,'花蓮縣':34.2,'新竹市':30.3,
    '屏東縣':23.1,'臺東縣':20.6,'臺中市':17.5,'臺北市':17.4,'新北市':16.7,
    '金門縣':16.5,'連江縣':16.0,'高雄市':14.7,'基隆市':12.5,'南投縣':12.3,
    '澎湖縣':9.5,'嘉義市':9.1,'雲林縣':8.5,'嘉義縣':8.3,'彰化縣':7.8,
    '宜蘭縣':7.7,'臺南市':7.1,
}
WAISHENG_COEF = {  # 眷村／都會係數
    '臺北市':1.5,'新竹市':1.6,'桃園市':1.4,'高雄市':1.3,'臺中市':1.3,'屏東縣':1.3,'基隆市':1.3,
    '新北市':1.0,'臺南市':0.9,'嘉義市':1.0,'宜蘭縣':0.9,'花蓮縣':0.9,'臺東縣':0.8,
    '雲林縣':0.5,'嘉義縣':0.4,'彰化縣':0.5,'南投縣':0.5,'苗栗縣':0.5,
    '澎湖縣':0.4,'金門縣':0.4,'連江縣':0.3,'新竹縣':0.9,
}

K = 0.55            # 客家縮放校準係數（單一認同目標 10.9%）
WAISHENG_BASE = 0.10

# 原住民法定比率：由戶政司表 2 讀入（此處示意，實務請 pd.read_excel 戶政開放資料）
ABORIGINAL = {  # 占縣市人口比率（小數）
    '臺東縣':0.381,'花蓮縣':0.303,'屏東縣':0.081,'桃園市':0.038,'新北市':0.016,
    '臺北市':0.008,  # ... 其餘縣市同樣由官方檔讀入
}

def county_shares(county):
    p_abo = ABORIGINAL.get(county, 0.01)
    p_hak = HAKKA_RAW.get(county, 0.0)/100 * K
    p_wai = WAISHENG_BASE * WAISHENG_COEF.get(county, 1.0)
    p_hok = 1 - p_abo - p_hak - p_wai
    if p_hok < 0:               # 客家高比例縣壓縮處理
        scale = (1 - p_abo) / (p_hak + p_wai)
        p_hak, p_wai, p_hok = p_hak*scale, p_wai*scale, 0.0
    return {'閩南':p_hok, '客家':p_hak, '外省':p_wai, '原住民':p_abo}

def assign_ethnicity(df, seed=42):
    rng = np.random.default_rng(seed)
    out = []
    for county, grp in df.groupby('居住縣市'):
        s = county_shares(county)
        labels, probs = list(s.keys()), np.array(list(s.values()))
        probs = probs / probs.sum()
        out.append(pd.Series(
            rng.choice(labels, size=len(grp), p=probs), index=grp.index))
    return pd.concat(out).reindex(df.index)

# === 套用 ===
# df['族群'] = assign_ethnicity(df)
```

## 7. 驗證步驟

1. **全國回算**：以各縣市人口為權重，加權各縣市族群比例，檢查全國閩南≈76–78%、客家≈11–14%、外省≈10%、原住民≈2.7%。
2. **原住民校驗**：花蓮、臺東 persona 的原住民比例須分別逼近 30.3%、38.1%。
3. **客家梯度校驗**：新竹縣 > 苗栗 > 桃園 > 花蓮的順序須保持。
4. **總和檢查**：每縣市四族比例和 = 1，無負值。

## 8. 已知限制與後續優化

| 限制 | 說明 | 優化方向 |
|------|------|---------|
| 外省縣市比例為推估 | 無官方縣市級外省統計，靠眷村係數代理 | 用國防部老舊眷村改建清冊、各縣市眷村列表逐縣校準 |
| 客家校準係數 `k` 為單一全國值 | 各縣市複數認同程度可能不同（客庄較高） | 若取得縣市別「單一認同/寬認同」比，可改逐縣係數 |
| 縣市層級抹平縣內差異 | 高雄左營眷村 vs 美濃客庄被平均 | 下沉至鄉鎮市區，套客家文化重點發展區（70 鄉鎮）與原住民族地區（55 鄉鎮）名單 |
| 整體比例引用較早期調查 | 主觀單一認同基準年份較舊 | 以最新國民生活狀況／社會變遷調查更新錨點 |
| 閩南為殘差 | 承接前三族所有推估誤差 | 可加獨立閩南認同調查交叉驗證 |
