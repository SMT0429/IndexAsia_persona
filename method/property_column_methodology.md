## 一、欄位定義

本欄位描述 persona 當前的住宅持有狀態，分為以下四類：

| 類別代碼 | 標籤 | 說明 |
|---------|------|------|
| A | 自有（本人或配偶名義） | 住宅所有權屬戶內經常居住成員 |
| B | 自有（家人名義） | 住宅為直系親屬（父母/子女）所有，本人居住其中 |
| C | 租屋 | 向他人承租，支付租金 |
| D | 借住 / 同住親屬 | 無須付租金之借住、眷村配住或寄住 |

> **定義來源：** 行政院主計總處《109年人口及住宅普查》普查表，住宅所有權屬分類：自有、租用、配住、借住（其他）。  
> 參見：[109年人口及住宅普查普查表](https://ws.dgbas.gov.tw/public/attachment/092910340fdxhdyf3.pdf)

## 二、資料基準：台北市底層機率分布

### 2.1 主要來源

| 數據 | 數值 | 來源 | 備註 |
|------|------|------|------|
| 台北市普通住戶**住宅自有率** | **72.1%** | 109年人口及住宅普查（主計總處，2021） | 直接引用，縣市別統計 |
| 全國普通住戶住宅自有率 | 78.6% | 同上 | 台北市低於全國 6.5pp |
| 全國住宅租用率 | 10.9% | 同上 | 台北市高於全國平均 |
| 台北市出租住宅單元佔比 | 13.8% | 政大不動產研究中心引用109年普查（2024） | 住宅單元口徑，非住戶口徑 |
| 全國自有住宅率（含直系親屬） | 84.4% | 113年家庭收支調查（主計總處，2024） | 較普查定義寬鬆 |

> **兩個「自有」定義的差異說明：**  
> - 普查「自有率 72.1%」= 住宅所有權屬**戶內成員**（嚴格）  
> - 家庭收支調查「84.4%」= 含**直系親屬**所有（寬鬆）  
> 差值（84.4% - 72.1% ≈ 12pp）大致對應本文分類 B「家人名義自有」的比例。

### 2.2 台北市底層機率估算

由於現有普查公布數字為台北市整體自有率（72.1%），租用與借住分類之縣市別明細尚須向主計總處申請微觀資料。本方法論採以下推算邏輯：

**步驟一：確定 A 類底層值**
```
A（本人名義自有）= 72.1%     [直接引用 109 年普查台北市縣市別數據]
```

**步驟二：估算 B 類（家人名義）**
```
家庭收支調查含親屬自有率 ≈ 84.4%（全國）
台北市普查自有率 72.1% 與全國 78.6% 差距 = -6.5pp
假設台北市「含親屬」自有率約 84.4% - 6.5% = 77.9%
∴ B = 77.9% - 72.1% = 5.8% ≈ 6%
```

**步驟三：估算 C、D 類**
```
全國非自有（含親屬）= 100% - 84.4% = 15.6%
台北市非自有（含親屬）≈ 100% - 77.9% = 22.1%
參照：台北市出租單元佔比 13.8%（政大，2024），租用戶口徑估 18-20%
∴ C（租屋）≈ 18%
∴ D（借住/配住）≈ 22.1% - 18% = 4.1% ≈ 4%
```

**台北市底層基準分布：**

| 類別 | 底層機率 | 信心程度 |
|------|---------|---------|
| A 自有（本人/配偶） | 72% | ★★★★★ 直接引用普查 |
| B 自有（家人名義） | 6% | ★★★☆☆ 推算 |
| C 租屋 | 18% | ★★★★☆ 普查+政大雙重支持 |
| D 借住/配住 | 4% | ★★★☆☆ 推算 |

## 三、條件調整方法

底層機率為台北市全體住戶的邊際分布。實際抽樣時，依 persona 的已知屬性乘上條件乘數（multiplier），再正規化為 1。

### 3.1 調整公式

```
P_raw(狀態 | 屬性組合) = P_base(狀態) × M_年齡 × M_收入 × M_婚姻
P(狀態) = P_raw(狀態) / Σ P_raw(所有狀態)   ← 正規化
```

### 3.2 年齡乘數（M_年齡）

文獻依據：109年普查提要分析顯示，台北市 65 歲以上老年人口比全國更集中，且高齡者與子女同住比例高；青年人口因房價所得比高，延後購屋。

| 年齡組 | A（自有本人） | B（家人名義） | C（租屋） | D（借住） | 文獻依據 |
|--------|-------------|-------------|---------|---------|---------|
| 15–24歲 | 0.05 | 1.8 | 2.2 | 1.8 | 多為學生/社會新鮮人、未婚、低收，本人名義持屋極罕見（QA issue 6） |
| 20–29歲 | 0.3 | 1.5 | 2.2 | 1.5 | 房價所得比高（台北逾16倍），政大 RER 2024 |
| 30–39歲 | 0.7 | 1.2 | 1.4 | 1.0 | 成家購屋高峰起點，婚姻帶動購屋 |
| 40–54歲 | 1.1 | 0.8 | 0.8 | 0.8 | 房貸還款中後期，自有率高峰 |
| 55–64歲 | 1.2 | 0.9 | 0.6 | 0.7 | 多數已完成購屋，子女漸獨立 |
| 65歲以上 | 1.2 | 1.0 | 0.4 | 1.2 | 高齡者部分與子女同住（借住分類提高） |

> 參見：[109年普查初步統計—高齡者居住概況](https://www.stat.gov.tw/public/Data/1112144316VT5YTOVB.pdf)，65歲以上與子女同住者占51.9%，獨居者15.6%。

### 3.3 月收入乘數（M_收入）

文獻依據：台灣房價所得比（政大不動產研究中心），台北市為全台最高；低收入戶以租屋或借住為主。

| 月收入區間 | A（自有本人） | B（家人名義） | C（租屋） | D（借住） | 邏輯 |
|----------|-------------|-------------|---------|---------|------|
| 4萬以下 | 0.45 | 1.25 | 1.75 | 1.5 | 無力購屋，多租屋或借住（原「未滿3萬」與「3~6萬」兩段平均） |
| 4~6萬 | 0.6 | 1.3 | 1.5 | 1.2 | 租屋為主，部分住家人房 |
| 6~10萬 | 1.0 | 1.0 | 1.0 | 0.7 | 接近平均值 |
| 10~15萬 | 1.2 | 0.8 | 0.7 | 0.5 | 有購屋能力 |
| 15~25萬 | 1.4 | 0.6 | 0.5 | 0.4 | 高自有率 |
| 25萬以上 | 1.4 | 0.6 | 0.5 | 0.4 | 高自有率 |

> 參見：[政大不動產研究中心—房價負擔能力統計](https://rer.nccu.edu.tw/article/detail/2210058908437)；[內政部住宅資訊平台](https://pip.moi.gov.tw/Publicize/Info/E1050)

### 3.4 婚姻狀況乘數（M_婚姻）

文獻依據：109年普查婚姻狀況數據顯示，35–49歲有偶者比率在60–70%，購屋與婚姻高度相關。

| 婚姻狀況 | A（自有本人） | B（家人名義） | C（租屋） | D（借住） | 邏輯 |
|---------|-------------|-------------|---------|---------|------|
| 已婚 | 1.3 | 0.9 | 0.7 | 0.6 | 家庭需求推動購屋 |
| 未婚 | 0.7 | 1.1 | 1.5 | 1.3 | 單身住需較小，傾向租屋 |
| 離婚/分居 | 0.8 | 1.0 | 1.4 | 1.0 | 分戶後重新進入租屋市場 |
| 喪偶 | 1.1 | 1.2 | 0.6 | 1.1 | 多繼承或同住子女 |

> 參見：[109年普查初步統計—婚姻狀況](https://www.stat.gov.tw/public/Data/1112144316VT5YTOVB.pdf) p.14–15

## 四、技術實作說明

### 4.1 函數簽名

```python
def sample_housing_status(
    age_group: str,       # 資料年齡組，e.g. '25–34歲'（內部映射至乘數鍵）
    income: str,          # 資料收入區間，e.g. '6~10萬'
    marital: str,         # e.g. '已婚'
    random_seed: int = None
) -> str:
    """
    回傳值：'自有（本人/配偶）' | '自有（家人名義）' | '租屋' | '借住/配住'
    """
```

### 4.2 實作邏輯

```python
import numpy as np

BASE = {
    '自有（本人/配偶）': 0.72,
    '自有（家人名義）': 0.06,
    '租屋':            0.18,
    '借住/配住':       0.04,
}

AGE_MULT = {
    '15–24歲': [0.05, 1.8, 2.2, 1.8],  # QA issue 6：青年本人名義持屋趨近 0
    '20–29歲': [0.3, 1.5, 2.2, 1.5],
    '30–39歲': [0.7, 1.2, 1.4, 1.0],
    '40–54歲': [1.1, 0.8, 0.8, 0.8],
    '55–64歲': [1.2, 0.9, 0.6, 0.7],
    '65歲以上': [1.2, 1.0, 0.4, 1.2],
}

INCOME_MULT = {
    # 資料標籤：4萬以下／4~6萬／6~10萬／10~15萬／15~25萬／25萬以上
    # '4萬以下' 跨越原「未滿3萬」與「3~6萬」兩段，取兩段平均
    '4萬以下':  [0.45, 1.25, 1.75, 1.5],
    '4~6萬':   [0.6, 1.3, 1.5, 1.2],
    '6~10萬':  [1.0, 1.0, 1.0, 0.7],
    '10~15萬': [1.2, 0.8, 0.7, 0.5],
    '15~25萬': [1.4, 0.6, 0.5, 0.4],
    '25萬以上': [1.4, 0.6, 0.5, 0.4],
}

MARITAL_MULT = {
    '已婚':   [1.3, 0.9, 0.7, 0.6],
    '未婚':   [0.7, 1.1, 1.5, 1.3],
    '離婚':   [0.8, 1.0, 1.4, 1.0],
    '喪偶':   [1.1, 1.2, 0.6, 1.1],
}

CATEGORIES = list(BASE.keys())

def sample_housing_status(age_group, income, marital,
                          random_seed=None):
    base = np.array([BASE[k] for k in CATEGORIES])
    
    # 年齡組映射（處理「45–54歲」→「40–54歲」等）
    age_key = map_age_group(age_group)
    
    am = np.array(AGE_MULT.get(age_key, [1,1,1,1]))
    im = np.array(INCOME_MULT.get(income, [1,1,1,1]))
    mm = np.array(MARITAL_MULT.get(marital, [1,1,1,1]))
    
    weights = base * am * im * mm
    
    # 正規化
    weights = weights / weights.sum()
    
    if random_seed is not None:
        np.random.seed(random_seed)
    
    return np.random.choice(CATEGORIES, p=weights)
```

### 4.3 年齡組映射函數

```python
def map_age_group(age_group: str) -> str:
    """將現有欄位年齡組映射至乘數表的年齡區間"""
    mapping = {
        '15–24歲': '15–24歲',  # QA issue 6：青年獨立鍵，A 乘數趨近 0
        '20–24歲': '15–24歲',
        '25–34歲': '20–29歲',  # 25–29 同屬青年
        '30–39歲': '30–39歲',
        '35–44歲': '30–39歲',  # 35–39 同屬
        '40–49歲': '40–54歲',
        '45–54歲': '40–54歲',
        '55–64歲': '55–64歲',
        '65歲以上': '65歲以上',
    }
    # 也可直接用年齡數值（若欄位有原始年齡）
    return mapping.get(age_group, '40–54歲')
```
