## 1. 欄位概述

| 項目 | 內容 |
|------|------|
| 欄位名稱 | `宗親地方組織連結強度` |
| 資料型態 | float，保留一位小數 |
| 值域 | 0.0 – 10.0 |
| 語意 | 分數越高，代表 persona 越可能嵌入宗親會、同鄉會、廟宇祭祀圈等地方人情網絡，選舉時受宗親動員影響的機率越高 |

### 與其他欄位的關係

本欄位是**衍生欄位**，由以下三個已存在欄位交叉計算產生：

- `族群`（閩南 / 客家 / 外省 / 原住民）
- `年齡組`（18–29 / 30–44 / 45–54 / 55–64 / 65歲以上）
- `居住地`（台北市 12 行政區）

---

## 2. 理論框架

本欄位建立在三個層次的推論之上：

### 2.1 族群差異（決定基礎分）

台灣宗親組織的形成與閩南移民的大姓制度高度相關。劉佩怡（2009）指出，台灣大姓（前百大姓）約占人口 95% 以上，閩南系大姓（陳、林、黃、張、李等）以「一姓多源」的方式組建宗親會，成為超越血緣的政治動員工具。客家族群則以同鄉會與義民祭祀圈為替代形式。外省族群的組織形態為「同籍同鄉會」（如湖南同鄉會、浙江同鄉會），制度功能類似但人情網絡隨世代快速衰退。原住民族則以部落制度為基礎，在都市環境中因地理分散導致部落連結顯著減弱。

**參考依據：**
- 劉佩怡，〈宗族、宗親會與選舉動員〉，《選舉研究》16(1)，2009，頁77–90。[華藝線上圖書館](https://www.airitilibrary.com/Article/Detail/19957548-200903-200911180007-200911180007-77-90)
- 吳重禮，〈台灣地區「派系政治」研究文獻的爭議：美國「機器政治」分析途徑的啟示〉，《政治科學論叢》第17期，2002，頁81–106。[全文PDF](https://ntupsr.s3.amazonaws.com/psr/wp-content/uploads/2011/10/17-04.pdf)
- 客家委員會，《110年全國客家人口暨語言基礎資料調查》，2021。全國客家人口約 466.9 萬人（占19.8%）；台北市客家人口比例低於全國均值，散居型態明顯。[客委會官網](https://www.hakka.gov.tw/chhakka/app/artwebsite?module=artwebsite&id=126)
- 原住民族委員會，〈臺灣都市原住民研究相關學位論文之回顧（1987–2015）〉，原住民族文獻，2015。指出都市原住民高度分散，部落連結機制依賴定期返鄉與城市部落組織，但普遍不如原鄉緊密。[原民會文獻](https://ihc.cip.gov.tw/EJournal/EJournalCat/343)

### 2.2 年齡差異（決定加權）

宗親會的社會連帶性（樁腳網絡與俗民網絡）仰賴長期累積的人情資本，因此年齡越大、嵌入程度越深。劉佩怡（2009）強調宗親組織的影響力建立在「長期經營關係與連結」之上。威權時代（1949–1987）國民黨透過農會、信合社、地方開發利益穩固地方派系，使當時成年的族群（現65歲以上）對宗親人情網絡有更強的制度依存。解嚴後成長的世代（現18–29歲）因都市化、網際網路及職場社群取代傳統組織，宗親認同顯著下降。

**參考依據：**
- 劉佩怡（2009），同上。
- 朱雲漢，〈寡佔經濟與威權政治體制〉，收錄於《威權主義的政治經濟學》，桂冠圖書，1992。說明國民黨如何以政治特權與寡佔經濟資源分配鞏固地方宗親網絡，使年長世代對這套制度的依存高於年輕世代。
- 內政部，《內政統計年報》（社會團體統計），歷年。可觀察宗教及宗親類社會團體會員數的年齡結構趨勢。[內政部統計](https://statis.moi.gov.tw/micst/webMain.aspx?k=menuy)

### 2.3 居住地差異（決定地域調整）

台北市1967年升格直轄市後，傳統地方派系（依賴鄉鎮市長選舉、農會信合社）的生存基礎大幅弱化，不存在如雲林「三派」或桃園「三派」的明確派系格局。然而，宗親/廟宇圈的影響仍以**歷史聚落**為單位留存。

- **萬華（艋舺）、大同（大稻埕、大龍峒）**：台北最早閩南移墾聚落（1709年陳賴章開墾），廟宇密度全市最高（龍山寺、霞海城隍廟、林氏宗廟等），宗親祭祀組織歷史最悠久。[台北市萬華區公所歷史沿革](https://whdo.gov.taipei/cp.aspx?n=CAA666D84C56861F)
- **士林、北投**：士林有閩南聚落（慈誠宮信仰圈）；北投則是台北市眷村數量最集中的行政區之一（中心新村等），外省族群同鄉網絡較強。
- **信義、松山**：信義區列管眷村達19處（四四東村、富臺新村等），是外省族群聚落密度最高的行政區。[台灣眷村列表 Wikipedia](https://zh.wikipedia.org/zh-tw/%E7%9C%B7%E6%9D%91%E5%88%97%E8%A1%A8)；[李廣均，〈臺灣「眷村」的歷史形成與社會差異〉，《臺灣社會學刊》57期，2015](https://www.tjs.org.tw/up_photo/moon-system/images/%E8%87%BA%E7%81%A3%E3%80%8C%E7%9C%B7%E6%9D%91%E3%80%8D%E7%9A%84%E6%B7%B1%E5%BA%A6%E5%BD%A2%E6%88%90%E8%88%87%E7%A4%BE%E6%9C%83%E5%B7%AE%E7%95%B0-%E5%88%97%E7%AE%A1%E7%9C%B7%E6%9D%91%E8%88%87%E8%87%AA%E5%8A%9B%E7%9C%B7%E6%9D%91%E7%9A%84%E6%AF%94%E8%BC%83.pdf)
- **南港、內湖**：1980年代後快速都市化的新興地區，居民流動性高，宗親聚落根基薄弱。

**參考依據：**
- 台北市政府，〈臺北歷史概述〉。[台北市政府官網](https://www.gov.taipei/cp.aspx?n=14D2637CBC4AAEC2&s=E0ACC486E3DA0A8C)
- 李廣均，〈臺灣「眷村」的歷史形成與社會差異〉，《臺灣社會學刊》第57期，2015，頁129–172。

## 3. 評分矩陣

### 3.1 第一層：族群基礎分

| 族群 | 基礎分 |
|------|--------|
| 閩南 | 5.0 |
| 客家 | 4.0 |
| 外省 | 2.5 |
| 原住民 | 2.0 |

### 3.2 第二層：年齡加權

| 年齡組 | 加分 |
|--------|------|
| 65歲以上 | +2.5 |
| 55–64歲 | +1.5 |
| 45–54歲 | +1.0 |
| 30–44歲 | +0.5 |
| 18–29歲 | −0.5 |

> 注意：現有資料的年齡組欄位為「45–54歲」與「55–64歲」需分開對應，65歲以上合併處理。

### 3.3 第三層：居住地加權

| 行政區 | 閩南／客家加分 | 外省加分 |
|--------|--------------|---------|
| 萬華區 | +1.5 | 0.0 |
| 大同區 | +1.5 | 0.0 |
| 士林區 | +1.0 | +1.0 |
| 北投區 | +0.5 | +1.5 |
| 信義區 | 0.0 | +1.5 |
| 松山區 | 0.0 | +1.0 |
| 中正區 | +0.5 | +1.0 |
| 文山區 | +0.5 | +0.5 |
| 大安區 | +0.5 | +0.5 |
| 中山區 | +0.0 | +0.5 |
| 南港區 | −0.5 | −0.5 |
| 內湖區 | −0.5 | −0.5 |

> 原住民族群一律使用閩南／客家欄的加分值（因非眷村關聯族群）。

### 3.4 壓縮公式

```
raw_score = 族群基礎分 + 年齡加分 + 居住地加分
final_score = round(min(max(raw_score, 0.0), 10.0), 1)
```

## 5. Python 實作

```python
import pandas as pd

# === 評分對照表 ===

ETHNIC_BASE = {
    '閩南': 5.0,
    '客家': 4.0,
    '外省': 2.5,
    '原住民': 2.0,
}

AGE_BONUS = {
    '65歲以上': 2.5,
    '55–64歲':  1.5,
    '45–54歲':  1.0,
    '30–44歲':  0.5,
    '18–29歲': -0.5,
}

# (行政區, 是否外省族群) → 加分
DISTRICT_BONUS = {
    ('萬華區', False): 1.5,  ('萬華區', True): 0.0,
    ('大同區', False): 1.5,  ('大同區', True): 0.0,
    ('士林區', False): 1.0,  ('士林區', True): 1.0,
    ('北投區', False): 0.5,  ('北投區', True): 1.5,
    ('信義區', False): 0.0,  ('信義區', True): 1.5,
    ('松山區', False): 0.0,  ('松山區', True): 1.0,
    ('中正區', False): 0.5,  ('中正區', True): 1.0,
    ('文山區', False): 0.5,  ('文山區', True): 0.5,
    ('大安區', False): 0.5,  ('大安區', True): 0.5,
    ('中山區', False): 0.0,  ('中山區', True): 0.5,
    ('南港區', False): -0.5, ('南港區', True): -0.5,
    ('內湖區', False): -0.5, ('內湖區', True): -0.5,
}


def calc_clan_score(row):
    ethnicity = row['族群']
    age_group = row['年齡組']
    district  = row['居住地']

    base      = ETHNIC_BASE.get(ethnicity, 3.0)
    age_adj   = AGE_BONUS.get(age_group, 0.0)
    is_waisheng = (ethnicity == '外省')
    dist_adj  = DISTRICT_BONUS.get((district, is_waisheng), 0.0)

    raw = base + age_adj + dist_adj
    return round(min(max(raw, 0.0), 10.0), 1)


# === 套用 ===
df = pd.read_excel('taipei_personas_3000_splitTicket.xlsx')
df['宗親地方組織連結強度'] = df.apply(calc_clan_score, axis=1)
df.to_excel('taipei_personas_3000_splitTicket.xlsx', index=False)

print(df['宗親地方組織連結強度'].describe())
print(df['宗親地方組織連結強度'].value_counts().sort_index())
```

## 6. 預期分布

基於台北市族群結構（閩南約 73%、客家約 8%、外省約 13%、原住民約 1%，依戶政司族群欄位實際分布為準）與年齡結構，預估分數分布如下：

| 分數區間 | 語意 | 預估占比 |
|---------|------|---------|
| 0.0–2.9 | 宗親網絡幾乎無連結 | ~10% |
| 3.0–4.9 | 低度連結（認同但不參與） | ~35% |
| 5.0–6.9 | 中度連結（偶有參與） | ~40% |
| 7.0–10.0 | 高度連結（積極參與、可被動員） | ~15% |

> 台北市整體分布預期偏向中低段（4–6分），反映直轄市特性——相較南部縣市或桃園，台北宗親動員的制度基礎較弱。

## 7. 已知限制與後續優化方向

| 限制 | 說明 | 優化方向 |
|------|------|---------|
| 無「世代」細分 | 外省第二代（年輕外省）同鄉連結遠低於第一代，目前靠年齡間接代理 | 可加入「政治與歷史印記_世代」欄位交叉調整 |
| 宗親會實際參與率缺乏直接統計 | 內政部社會團體資料未獨立列出「宗親會」類別 | 可用中選委里層得票資料逆推估 |
| 未納入婚姻狀況 | 已婚者嵌入地方社群機率較高，未加入以控制複雜度 | 已婚可加+0.3作為次要選項 |
| 地域加分基於歷史聚落推論 | 實際里層宗親組織分布未有完整公開資料 | 可參考台北市文獻委員會地名沿革報告補充 |
