## 四、非就業人口判斷邏輯

依據 persona 現有 `職業` 欄位值判斷是否為就業人口：

```python
NON_EMPLOYED = {'退休', '學生', '家管', '其他/待業'}

def get_industry(persona):
    occ = persona['職業']
    if occ in NON_EMPLOYED:
        return '無（非就業人口）'
    elif occ == '農林漁牧':
        return '農林漁牧業'   # 直接指定，無隨機抽樣
    else:
        return sample_by_occupation(occ, persona)
```

**特殊規則：**
- `農林漁牧`（職業）→ 直接指定 `農林漁牧業`（行業），無需抽樣
- `其他/待業` 歸入非就業，不分配產業（視為勞動力邊緣人口）

## 五、職業別條件機率矩陣

### 5.1 設計說明

表46 提供的是行業 × 年齡/教育的交叉表，**未直接提供行業 × 職業**的聯合分布。  
本矩陣依據以下依據設計：

1. **勞動市場結構推論**：依主計總處行業×職業理論對應關係（標準職業分類與行業分類相關性）
2. **台北市產業特性**：對台北市偏高產業（金融、資訊、專業技術）加權
3. **學術文獻參照**：Heckman & Honoré (1990) 職業排序模型；台灣就業市場相關研究

矩陣值為**相對權重（unnormalized weight）**，抽樣前需正規化為機率分布。

### 5.3 台北市調整（Taipei Adjustment）

全國矩陣之後，對台北市 persona 套用行業層級調整係數（multiplier），再重新正規化：

```python
TAIPEI_MULTIPLIER = {
    '農林漁牧業':        0.05,  # 台北市幾乎無農業
    '製造業':            0.25,  # 台北市製造業遠低於全國
    '工程及公用事業':    0.60,  # 略低
    '批發及零售業':      1.10,  # 略高
    '住宿及餐飲業':      1.00,  # 持平
    '運輸及倉儲業':      0.80,  # 略低（無工業港口）
    '資訊及通訊傳播業':  3.00,  # 台北市資訊業佔比顯著高於全國
    '金融及保險業':      2.50,  # 台北為金融中心
    '不動產及專業技術業':1.50,  # 偏高
    '公共行政及國防':    1.80,  # 首都效應
    '教育業':            1.30,  # 大學密集
    '醫療保健及社會工作':1.20,  # 醫學中心密集
    '其他服務業':        1.00,  # 持平
}
```

> **調整係數來源說明：**  
> 係數根據臺北市政府主計處發布之「臺北市就業人口之行業」PDF（統計年報第34表）、  
> 及臺北產經資訊網「臺北產經動態」季報（2023Q4、2024Q4）中產業結構比較推估。  
> 若取得原始表34數字後，應以實際比例取代本係數，並重新計算矩陣。

## 六、抽樣演算法

```python
import random

def sample_by_occupation(occupation: str, persona: dict) -> str:
    """
    依職業類別條件機率抽樣行業。
    
    Args:
        occupation: 職業欄位值（已排除非就業人口與農林漁牧）
        persona: 完整 persona dict（備用，可擴充年齡交叉加權）
    
    Returns:
        行業大分類字串
    """
    # 1. 取得該職業的原始權重向量
    raw_weights = OCCUPATION_INDUSTRY_MATRIX[occupation]
    # raw_weights: dict {行業名稱: 相對權重}

    # 2. 套用台北市調整係數
    adjusted_weights = {
        industry: raw_weights[industry] * TAIPEI_MULTIPLIER[industry]
        for industry in INDUSTRIES
    }

    # 3. 正規化為機率分布
    total = sum(adjusted_weights.values())
    probs = {k: v / total for k, v in adjusted_weights.items()}

    # 4. 加權隨機抽樣
    industries = list(probs.keys())
    weights = list(probs.values())
    return random.choices(industries, weights=weights, k=1)[0]


def assign_industry(persona: dict) -> str:
    """主函式：根據 persona 的職業欄位指派產業"""
    occ = persona['職業']
    
    if occ in NON_EMPLOYED:
        return '無（非就業人口）'
    elif occ == '農林漁牧':
        return '農林漁牧業'
    else:
        return sample_by_occupation(occ, persona)
```
