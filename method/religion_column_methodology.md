### 4.3 抽樣演算法

```python
import random

CATEGORIES = ['傳統民間信仰', '無宗教', '佛教', '道教', '基督新教', '一貫道', '天主教', '其他宗教']

BASE_PROB = {
    '傳統民間信仰': 0.236,
    '無宗教':       0.308,
    '佛教':         0.196,
    '道教':         0.148,
    '基督新教':     0.065,
    '一貫道':       0.022,
    '天主教':       0.015,
    '其他宗教':     0.006,
}

def get_age_group(age: int) -> str:
    if age <= 35:
        return '15-35'
    elif age <= 55:
        return '36-55'
    else:
        return '56+'

def assign_religion(persona: dict) -> str:
    """
    依年齡條件機率加權抽樣宗教類別。
    
    Args:
        persona: 包含 '年齡' 欄位的 persona dict
    
    Returns:
        宗教與地方信仰類別字串
    """
    age_group = get_age_group(persona['年齡'])
    multipliers = AGE_MULTIPLIER[age_group]

    # 套用調整係數
    adjusted = {cat: BASE_PROB[cat] * multipliers[cat] for cat in CATEGORIES}

    # 正規化
    total = sum(adjusted.values())
    probs = {k: v / total for k, v in adjusted.items()}

    # 加權隨機抽樣
    return random.choices(
        list(probs.keys()),
        weights=list(probs.values()),
        k=1
    )[0]
```

## 七、方法論限制

1. **無台北市別宗教自報調查**：內政部宗教資訊網以**寺廟登記數**統計，非信徒自報比例，城市層級的自報分布須依全國調查（TSCS）推估，存在估計誤差。

2. **類別邊界模糊**：台灣民眾普遍有「有拜有保庇」的複合信仰行為，佛道混合比例高（AIT 報告指出多達 80% 的信仰者同時混合多種傳統）。本欄位採排他性主要信仰口徑，部分民眾的實際信仰可能跨越多個類別，分類結果為統計估算而非個人精確自報。

3. **世代效應快速變動**：台灣世俗化趨勢明顯（2024 普查顯示基督徒比例持續下降），基準比例應定期更新。

4. **宮廟動員研究以質性研究為主**：宮廟與地方派系動員的相關文獻多為田野研究，缺乏全國量化調查數據，本欄位對動員力的描述屬定性標記，非統計精確估計。
