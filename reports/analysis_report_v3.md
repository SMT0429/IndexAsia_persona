### 2.8 國家認同

來源：民調 Q2～Q5，詢問受訪者對政府在**經濟發展、能源政策、低薪高房價、司法公正**四項議題的信心程度。

**計算方式：**

| 步驟 | 說明 |
|---|---|
| 原始量表 | 1=非常有信心、2=還算有信心、3=不太有信心、4=完全沒有信心 |
| 反轉 | `inverted = 5 - value`，使高分代表高信任（1→4, 4→1） |
| 取平均 | 對四題中回答有效（1–4）者取平均值；96（不知道）、98（拒答）排除不計 |
| 標準化 | `score = (mean − 1) / 3 × 10`，輸出範圍 **0–10 分** |

**問卷參考分布（n=1,099）：**

| 年齡組 | 平均信任指數 |
|---|---:|
| 15–24 歲 | 4.15 |
| 25–34 歲 | 3.49 |
| 35–44 歲 | 4.23 |
| 45–54 歲 | 4.51 |
| 55–64 歲 | 3.87 |
| 65 歲以上 | 5.27 |
| **全體** | **4.53** |

> 高齡者信任度最高（5.27），25–34 歲最低（3.49），反映世代間政治態度差異。

## 三、抽樣方法

### 3.1 抽樣流程

每一筆 persona 依序抽樣，各維度條件獨立或依條件分布：

```
for i in 1..3000:
    居住地  ← 依縣市人口比例（22 縣市加權，每縣市保底 1 筆）
    性別    ← 依該縣市男女比例（條件於居住地）
    年齡    ← 依 15 歲以上 5 歲組人口比例，組內均勻分布至具體年齡
    教育程度 ← 依該縣市 15+ 教育程度比例（條件於居住地）
    職業    ← 條件於年齡（非就業機率）+ 條件於居住地（就業者職業結構）
    婚姻狀況 ← 條件於具體年齡（有偶率查表，5 歲組）
    媒體習慣 ← 條件於年齡組，各平台獨立 Bernoulli 抽樣
    政治維度 ← 依人口特徵分層比對民調子樣本（詳見 3.3）
```

**隨機種子：** `seed = 42`（確保結果可重現）

### 3.2 條件依賴關係

```
居住地 ──→ 性別
居住地 ──→ 教育程度
居住地 ──→ 職業（就業者結構）
年齡   ──→ 職業（非就業機率）
年齡   ──→ 婚姻狀況（有偶率）
年齡組 ──→ 媒體習慣（各平台使用率）
居住地 + 年齡組 + 教育 + 性別 ──→ 政治三維度（分層比對）
```

### 3.3 政治維度分配方法

政治三維度（國家認同、政黨傾向、厭惡政黨）以**人口特徵分層比對**的方式，將民調樣本的回答分布映射至 3,000 筆 persona。

**比對邏輯：**

對每一筆 persona，依以下 8 層優先順序在民調資料中尋找「人口特徵相近」的子樣本，要求子樣本 ≥ 10 筆才採用，否則逐層放寬：

| 優先層 | 比對條件 |
|---|---|
| 1 | 縣市 + 年齡組 + 教育程度 + 性別 |
| 2 | 縣市 + 年齡組 + 性別 |
| 3 | 縣市 + 年齡組 |
| 4 | 大區（北 / 中 / 南 / 東 / 離島）+ 年齡組 + 教育程度 + 性別 |
| 5 | 大區 + 年齡組 + 性別 |
| 6 | 大區 + 年齡組 |
| 7 | 年齡組 + 性別 |
| 8 | 年齡組 |
| Fallback | 全體民調樣本（n=1,099） |

**大區定義：**

| 大區 | 縣市 |
|---|---|
| 北部 | 新北、臺北、桃園、基隆、新竹市、新竹縣、宜蘭 |
| 中部 | 臺中、苗栗、彰化、南投、雲林 |
| 南部 | 臺南、高雄、嘉義市、嘉義縣、屏東、澎湖 |
| 東部 | 臺東、花蓮 |
| 離島 | 金門、連江 |

**各維度取樣方式：**

- **國家認同**：從比對子樣本中隨機抽取一個信任分數，加入常態擾動（σ=0.25），clip 至 [0, 10]，輸出至小數點後一位。
- **政黨傾向**：從比對子樣本的 Q12 有效回答（包含「不知道」，排除 NaN）中均勻隨機抽取。
- **厭惡政黨**：從比對子樣本的 Q13 有效回答中抽取，**若 persona 已有明確政黨偏好（民進黨 / 國民黨 / 台灣民眾黨），則從池中排除該黨後再抽樣**，確保政黨傾向與厭惡政黨不互相矛盾。

**邏輯一致性約束：**

```
IF 政黨傾向 ∈ {民進黨, 國民黨, 台灣民眾黨}
THEN 厭惡政黨 ≠ 政黨傾向
```

排除後若剩餘至少 1 筆有效回答則使用過濾後的池；否則回退至原池（實際上無此情況）。  
**最終衝突筆數：0 / 3,000**。

### 3.4 資料整合說明

- **金門縣、連江縣**：無縣市層級職業資料，改用全台就業結構
- **連江縣保底機制**：縣市人口比例極低（約 0.1%），採用「每縣市保底 1 筆 + 剩餘 2,978 筆依比例分配」確保全 22 縣市均有代表
- **婚姻狀況**：僅輸出「已婚（有偶）/ 未婚」；離婚、喪偶均歸入未婚
- **媒體習慣**：NCC 調查年齡組為 16–25 歲起，15 歲者套用同組數值；15 個平台各自獨立 Bernoulli 抽樣後，**無條件強制保底 LINE**（抽樣結果若不含 LINE 則強制插入，因 LINE 在台灣滲透率近全民 >90%），故輸出中 LINE 佔比為 100%；65 歲以上強制排除 Instagram、TikTok、小紅書、Threads、X (Twitter)、LinkedIn（該組使用率 < 2%）
- **政治維度**：民調 15–19 歲組無資料，15–24 歲 persona 以 20–24 歲問卷回答代替

## 六、資料限制與說明

1. **職業非就業類別**：家管、學生、退休無直接政府統計，以年齡條件機率估算，非實際普查數字。
2. **媒體習慣**：NCC 調查對象為社群媒體「使用者」，非全體國民，可能低估高齡不上網族群。
3. **婚姻狀況**：僅區分已婚（有偶）與未婚，離婚與喪偶均歸未婚，與實際婚史有差異。
4. **教育程度**：以各縣市當下在住人口為基礎，不反映年齡內的教育差異（老一輩教育程度較低）。
5. **金門縣、連江縣**：職業分布採全國平均替代，因原始統計未涵蓋離島縣市職業細項。
6. **國家認同**：採隨機抽樣加擾動（σ=0.25）生成，非直接插值；生成均值（4.26）較問卷參考值（4.53）略低，屬擾動效應，各年齡組趨勢方向一致。
7. **厭惡政黨排除效應**：因邏輯約束（偏好黨 ≠ 厭惡黨），偏好民進黨的 persona（約 42%）被排除「民進黨」選項，導致「民進黨」在厭惡政黨中的生成比例低於問卷原始比例（-8.8%），差距不代表抽樣誤差。
8. **政治民調代表性**：民調 n=1,099，以電話訪問進行，高齡受訪者佔比偏高（70 歲以上佔 32.8%），可能低估年輕世代政治態度的多樣性。
