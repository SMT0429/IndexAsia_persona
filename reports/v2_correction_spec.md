## Issue 1：政黨傾向 ↔ 國家認同 不一致

### 1.1 問題現況

| 矛盾類型 | 筆數 | 占該黨總數 |
|---------|------|----------|
| 民進黨 傾向 但 國家認同 ≤ 2.0（強親中） | 275 / 930 | **29.6%** |
| 國民黨 傾向 但 國家認同 ≥ 8.0（強台獨） | 115 / 1,047 | **11.0%** |

目前各黨國家認同分布：

| 政黨 | 平均值 | 中位數 |
|------|--------|--------|
| 民進黨 | 4.46 | 4.8 |
| 國民黨 | 4.15 | 4.3 |
| 台灣民眾黨 | 4.35 | 4.7 |

四黨的平均值幾乎相同，顯示兩個欄位是**獨立隨機生成**，沒有聯合分布約束。

### 1.2 實際數據基準

依台灣選舉與民主化調查（TEDS）及政大選研中心 2020–2024 年數據：

| 政黨選民 | 國家認同平均分（0=統一、10=獨立）| 落在 0–2 的比例 | 落在 8–10 的比例 |
|---------|-------------------------------|--------------|--------------|
| 民進黨 | 約 7.2–7.8 | < 3% | > 50% |
| 國民黨 | 約 4.5–5.5 | 約 15–20% | < 5% |
| 台灣民眾黨 | 約 5.5–6.5 | 約 8–12% | 約 15–25% |
| 其他政黨 | 約 5.0–6.0 | — | — |

### 1.3 修正規則

生成 `國家認同`（0–10 浮點數）時，依 `政黨傾向` 套用以下分布：

```
民進黨      → 常態分布 N(μ=7.2, σ=1.4)，截斷於 [3.0, 10.0]
國民黨      → 常態分布 N(μ=5.0, σ=2.0)，截斷於 [0.0, 8.5]
台灣民眾黨  → 常態分布 N(μ=6.0, σ=2.0)，截斷於 [1.0, 10.0]
其他政黨    → 均勻分布 U(2.0, 9.0)
```

### 1.4 驗收標準

修正後需滿足：

- 民進黨選民 國家認同 ≤ 2.0 的比例 < **3%**
- 國民黨選民 國家認同 ≥ 8.0 的比例 < **5%**
- 各黨平均值差距 ≥ **1.0 分**（民進黨 >> 台灣民眾黨 > 國民黨）

## Issue 3：媒體習慣（WeChat / 小紅書）↔ 國家認同 不一致

### 3.1 問題現況

| 問題 | 筆數 | 占該平台使用者比例 |
|------|------|----------------|
| WeChat 使用者 且 國家認同 ≥ 8.0 | 45 / 415 | **10.8%** |
| 小紅書使用者 且 國家認同 ≥ 8.0 | 47 / 482 | **9.8%** |

目前 WeChat 使用者平均國家認同 3.98、小紅書使用者 4.23，整體略低，但 8.0 以上的尾端仍有約 10% 的矛盾。

### 3.2 實際數據基準

WeChat 與小紅書在台灣的使用者輪廓（政大傳播學院、台大社會所調查）：

- 主要使用族群：有兩岸商業往來、在中國有家人或工作、或中國籍在台人士
- 台灣本土強認同者（國家認同 ≥ 7）使用 WeChat 或 小紅書的比例 **< 5%**
- 使用 WeChat / 小紅書的用戶，其國家認同分布明顯偏低（平均估計約 3.5–4.5）

### 3.3 修正規則

在媒體習慣組合中包含 WeChat 或 小紅書 時，對應的 `國家認同` 需套用上限約束：

```
若 媒體習慣 包含 WeChat  → 國家認同 ∈ [0.0, 6.5]
若 媒體習慣 包含 小紅書  → 國家認同 ∈ [0.0, 7.0]
兩者都有時               → 國家認同 ∈ [0.0, 5.5]
```

或反向：若先生成 `國家認同 ≥ 7.5`，則在媒體習慣候選池中移除 WeChat 和小紅書。

### 3.4 驗收標準

- WeChat 使用者 國家認同 ≥ 8.0 的比例 < **3%**
- 小紅書使用者 國家認同 ≥ 8.0 的比例 < **3%**
- WeChat 使用者平均國家認同 ≤ **4.5**

## Issue 4：職業（家管）↔ 婚姻狀況 / 性別 不一致

### 4.1 問題現況

| 問題 | 筆數 | 占家管總數（345） |
|------|------|----------------|
| 男性家管 | 160 | **46.4%** |
| 未婚家管 | 159 | **46.1%** |
| 男性且未婚家管 | — | — |

台灣家管的真實性別與婚姻構成與此差距極大。

### 4.2 實際數據基準

依主計總處「人力資源調查」（2023 年平均）：

| 類別 | 真實比例 |
|------|---------|
| 家管中女性佔比 | **92–94%** |
| 家管中男性佔比 | **6–8%** |
| 家管中已婚佔比 | **88–92%** |
| 家管中未婚佔比 | **8–12%**（多為喪偶或離婚後照顧子女） |

### 4.3 修正規則

生成 `職業 = 家管` 的 agent 時，套用以下條件機率：

```
性別分布：女性 90%，男性 10%
婚姻分布：已婚 88%，未婚 12%
```

**額外限制**：
- 男性家管 且 未婚 的組合比例 < **3%**（現實中極少見）
- 家管的年齡應集中於 30–65 歲（主要照顧子女或老人時期）

### 4.4 驗收標準

- 家管中女性比例 ≥ **85%**
- 家管中已婚比例 ≥ **80%**
- 男性未婚家管 < **5 筆**（3,000 筆資料中）

## Issue 5：年齡 ↔ 職業（高齡在職）不一致

### 5.1 問題現況

| 年齡區間 | 總筆數 | 仍在職（非退休/家管）| 比例 |
|---------|--------|-------------------|------|
| 70 歲以上 | 528 | 76 | **14.4%** |
| 75 歲以上 | 305 | 40 | **13.1%** |
| 80 歲以上 | 164 | 20 | **12.2%** |

75 歲以上仍在職者職業分布：
- 其他/待業：12 筆
- 服務/銷售：7 筆
- 事務人員：7 筆
- 專業人員：6 筆
- 技術工/勞工：5 筆

包含 97 歲碩士專業人員、82 歲服務業人員、80 歲技術工等極端案例。

### 5.2 實際數據基準

依主計總處「人力資源調查」及勞動部統計（2023）：

| 年齡區間 | 勞動參與率（實際） |
|---------|----------------|
| 65–69 歲 | 約 **16–20%**（多為自營業或兼職） |
| 70–74 歲 | 約 **8–10%** |
| 75 歲以上 | 約 **3–5%** |
| 80 歲以上 | < **1%** |

台灣法定退休年齡 65 歲；75 歲以上仍從事全職正規職業者在都市中（台北）更是罕見。

### 5.3 修正規則

依年齡設定職業可選範圍：

```
65–69 歲：退休機率 75%；其他職業機率 25%（限：專業人員、管理/主管、自雇性質）
70–74 歲：退休機率 90%；其他職業機率 10%（限：專業人員）
75 歲以上：退休機率 97%；其他職業機率 3%（僅保留退休、家管）
80 歲以上：強制為 退休 或 家管
```

### 5.4 驗收標準

- 75 歲以上仍在職（非退休/家管）比例 < **4%**（約 12 筆以內）
- 80 歲以上非退休/家管：**0 筆**

## 建議生成順序（Dependency Graph）

Issue 之間存在欄位依賴，建議生成順序如下：

```
1. 年齡 → 年齡組
2. 年齡 → 職業（加入高齡限制）
3. 職業 → 性別、婚姻狀況（家管條件機率）
4. 政黨傾向 → 國家認同（聯合分布）
5. 年齡 → 媒體習慣候選池
6. 國家認同 → 媒體習慣（移除 WeChat/小紅書）
```

Issue 1、3、4 的核心問題都是**欄位獨立生成後未做交叉驗證**，建議改為條件生成（conditional generation）而非各欄獨立抽樣。
