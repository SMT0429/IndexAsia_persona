## 三、抽樣方法

### 3.1 抽樣流程

每一筆 persona 依序抽樣，隨機種子固定為 **seed = 42**，確保結果完全可重現。共抽 3,000 筆，涵蓋臺北市 12 行政區，各區保底至少 1 筆後依人口比例分配剩餘配額。

```
for i in 1..3000:
    居住地  ← 依各行政區人口比例（保底每區 1 筆）
    性別    ← 各區男女配額精確分配（非逐筆 Bernoulli，避免 RNG 序列偏差）
    年齡    ← 依各區 5 歲組人口比例，組內均勻分布至具體年齡
    教育程度 ← 依各行政區 15+ 教育程度比例（條件於居住地）
    職業    ← 條件於年齡（非就業機率）+ 臺北市就業者職業結構
    婚姻狀況 ← 條件於具體年齡（全台有偶率）× 行政區婚姻密度調整因子
    媒體習慣 ← 條件於年齡組，各平台獨立 Bernoulli 抽樣；零平台時強制補入 LINE
    政黨傾向 ← 條件於居住地：依各區 113 年選舉得票比例（4 黨）
    國家認同 ← 從分層民調子樣本隨機抽值，加入常態擾動（σ=0.25），clip 至 [0, 10]
    厭惡政黨 ← 同上子樣本，若偏好黨 ∈ {民進黨, 國民黨, 台灣民眾黨} 則排除後再抽
```

---

### 3.2 條件依賴關係

```
居住地 ──→ 性別
居住地 ──→ 年齡（5歲組分布）
居住地 ──→ 教育程度
居住地 ──→ 政黨傾向（選舉得票率）
居住地 ──→ 婚姻狀況（婚姻密度調整因子）
年齡   ──→ 職業（非就業機率）
年齡組 ──→ 媒體習慣（各平台使用率）
年齡組 + 教育 + 性別 ──→ 國家認同、厭惡政黨（分層比對）
```

---

### 3.3 政治維度分層比對方法

國家認同與厭惡政黨以民調資料的**人口特徵分層比對**方式產生，對每一筆 persona 依以下 8 層優先順序尋找民調子樣本（要求 ≥10 筆），不足則逐層放寬：

| 優先層 | 比對條件 |
|---|---|
| 1 | 臺北市 + 年齡組 + 教育程度 + 性別 |
| 2 | 臺北市 + 年齡組 + 性別 |
| 3 | 臺北市 + 年齡組 |
| 4 | 北部（含宜蘭/新竹/桃園等）+ 年齡組 + 教育程度 + 性別 |
| 5 | 北部 + 年齡組 + 性別 |
| 6 | 北部 + 年齡組 |
| 7 | 年齡組 + 性別 |
| 8 | 年齡組 |
| Fallback | 全體民調樣本（n=1,099） |

**各維度取樣方式：**

- **國家認同**：從比對子樣本中隨機抽取一個信任分數，加入常態擾動（σ=0.25），clip 至 [0, 10]，輸出至小數點後一位。
- **厭惡政黨**：從比對子樣本的 Q13 有效回答中抽取，**若 persona 已有明確政黨偏好（民進黨 / 國民黨 / 台灣民眾黨），則從池中排除該黨後再抽樣**，確保政黨傾向與厭惡政黨不互相矛盾。

**邏輯一致性約束：**
```
IF 政黨傾向 ∈ {民進黨, 國民黨, 台灣民眾黨}
THEN 厭惡政黨 ≠ 政黨傾向
```

排除後若剩餘至少 1 筆有效回答則使用過濾後的池，否則回退至原池。**最終衝突筆數：0 / 3,000**。

| 項目 | 說明 |
|---|---|
| 政黨傾向資料 | 臺北版改用選舉數據（4 黨分類），不含「不偏任何黨」、「不知道」等問卷選項 |
| 婚姻調整因子 | 以城市整體婚姻率（13,124 對 / 2,187,844 人）為基準，各區比值 clip 至 [0.6, 1.5] |

| 政治民調年齡 | 15–19 歲組無民調資料，以 20–24 歲問卷回答代替 |

## 六、資料限制與說明

1. **政黨傾向與問卷定義不同**  
   臺北版使用選舉得票資料（民進黨 33.4%、國民黨 37.5%、民眾黨 21.8%），與問卷偏好（Q12）定義不同，無法直接比較。選舉數據反映「實際投票行為」，問卷偏好反映「政黨認同」，兩者皆為合理的政黨傾向衡量方式。

2. **政黨傾向無「不偏任何黨」類別**  
   由於臺北版依據選舉數據分類，輸出的政黨傾向僅包含民進黨、國民黨、台灣民眾黨、其他政黨四類，不含「不偏任何黨」或「不知道」。

3. **職業非就業類別**  
   家管、學生、退休無直接對應政府統計，以年齡條件機率估算，非實際普查數字。

4. **媒體習慣調查偏差**  
   NCC 調查對象為社群媒體「使用者」，非全體國民，可能低估高齡不上網族群的使用率。

5. **婚姻狀況簡化**  
   僅區分已婚（有偶）與未婚，離婚與喪偶均歸未婚，與實際婚史有差異。

6. **教育程度年齡異質性**  
   以各行政區當下在住人口為基礎，不反映年齡內的教育差異（老一輩教育程度較低）。

7. **國家認同民調代表性**  
   民調 n=1,099，以電話訪問進行，高齡受訪者佔比偏高（70歲以上佔 32.8%），可能低估年輕世代政治態度的多樣性。

8. **地理精確度**  
   本版 JSON 輸出中「居住地」欄位統一標記為「臺北市」，行政區資訊保存於 Excel 版本；如需行政區層級分析，請使用 `taipei_personas_3000.xlsx` 的 Personas 工作表。
